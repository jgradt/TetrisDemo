# TetrisDemo
Simple demo of Tetris running in the browser.  

Some of the technologies involved include:
- [Typescript](http://www.typescriptlang.org)
- [Reactive Extensions for Javascript (RxJS)](http://reactivex.io/rxjs/)
- [Webpack](https://webpack.js.org/)
- [Gulp](https://gulpjs.com/)

# Local Setup:

Initial Setup:
- Download or clone the repository and open it in [Visual Studio Code](https://code.visualstudio.com/).  
- Restore the npm packages locally.  
- You may need to make sure gulp is installed locally and globally
- Run the gulp task "build-run-watch"
- In the "Debug" window, press the green arrow to "Launch localhost in Chrome"
